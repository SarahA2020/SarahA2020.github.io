My Portfolio

Welcome to my projects. Please click on the individual project cards below to browse through some of the different builds I have been working on.

Collection App - PHP

This application allows the user to view a collection of football player cards and add their own cards via the form provided. The project uses unit-tested PHP to store and retrieve data from a SQL database. The front end is built with HTML and CSS and utilises both flex and grid layouts. Implements a responsive design.

Maze Game - JavaScript

An interactive maze game with a ghoulish theme built within a Scrum team of 8. JavaScript is used to start the game timer, animate obstacles, and trigger winning or losing modals depending on specific DOM events.

Ecommerce Site - PHP

This group project takes an OOP approach and implements SOLID programming principles. PHP is used to communicate with a SQL database, populating it with JSON data imported from an API, then displaying to the user.

Watermelon - SVG

A bit of fun, this mini project indulges my creative side and gave me some great practice using svg paths and arcs.
